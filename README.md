## 🚀 Как помочь проекту?

Если у тебя есть пользователи, которых нет в списке (абузили триал, скамили, шерили подписки и т.д.):

1. **Вступай в группу**: [Bedolaga Social Club](https://t.me/+wTdMtSWq8YdmZmVi)
2. **Оформляй репорт** новых пользователей в теме **Черный список**:

    * Username (при наличии)
    * Причина бана
